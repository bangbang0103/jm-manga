# JM Manga Flutter Client

Flutter client for JM Manga. Currently supports Android and iOS only.

Common commands:

```bash
flutter pub get
flutter analyze --fatal-infos
flutter test
flutter run
```

Release and build scripts are located at `../scripts/release.sh`, `../scripts/build-flutter.sh` and `../scripts/package-server.sh`.
